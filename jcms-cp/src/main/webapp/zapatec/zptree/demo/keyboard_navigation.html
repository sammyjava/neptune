Keyboard navigation
  • Use keyboardNavigation config option to turn on keyboad navigation for tree. Following keys are avaiable:
    • Up arrow - previous node.
    • Down arrow - next node.
    • Left arrow - parent node/collapse branch.
    • Right arrow - expand branch/go to first child.
    • Enter/spacebar - toggle branch visibility.
    • Esc - leave tree (disable navigation).
  • Using deselectOnLeave config option you can define, that selected node must be deselected if user leaves tree.
  • User can leave tree (keyboard navigation will be disabled) by pressing Esc key or clicking somewhere at the page outside tree.
  • It is impossible to use keyboardNavigation togeteher with selectMultiple. If selectMultiple config option is true - keyboardNavigation config option will be ignored.
  • If keyboardNavigation is true and deselectOnLeave is not defined explicitly - it will be treated as true.
