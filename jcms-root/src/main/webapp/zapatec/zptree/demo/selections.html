Customize node selection in the tree
  • This demo shows how tree can be used to select data from the tree using selection
  • selectMultiple:true config option tells that tree can have more than one selected node at a time
  • deselectSelected:true config option tells that when you click on already selected node - it will be deselected
  • selectOnSignClick: false, selectOnSignDblclick: false, selectOnIconClick: false, selectOnIconDblclick: false allows node selection only on label click.
