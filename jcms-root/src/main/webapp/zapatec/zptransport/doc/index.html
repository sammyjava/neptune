The Zapatec Transport Overview    
The Zapatec Transport provides communication between the web browser and the web server without refreshing the page. It also provides parsing and utility methods to facilitate sending data to and fetching data from the server.

Start by checking out the Transport demo and see the various ways the Zapatec Transport can be used.
