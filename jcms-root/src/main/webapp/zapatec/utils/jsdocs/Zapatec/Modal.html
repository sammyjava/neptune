Constructor Detail

Zapatec.Modal

Zapatec.Modal(config)
    Zapatec modal object. It represents the element that imitates modal behaviour. The constructor recognizes the following options: \code prop. name | description ------------------------------------------------------------------------------------------------- zIndex | {number} z-index for the element. Default 1000. x | {number} X coordinate of the element. Default null. y | {number} Y coordinate of the element. Default null. width | {number} width of the element. Default null. height | {number} height of the element. Default null. container | {HTML element or Default View} container for the element. Default window. fixed | {boolean} if the element is fixed on screen. Default false. \endcode
