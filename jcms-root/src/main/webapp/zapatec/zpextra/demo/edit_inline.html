Edit inline
  • This demo shows how to use Zapatec.EditInline widget. It allows you to edit text in elements on the page
  • You can show edit window in 2 ways
    • call Zapatec.EditInline constructor and send reference to element as target config option (click at "First" and "Second" to see this)
    • create instance of Zapatec.EditInline and call show() method. It accepts reference to target element as first argument.(click at "Third" and "Fourth" to see this)
