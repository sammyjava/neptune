Editable tree
  • Setting editable config option to true you define that tree nodes can be edited.
  • editOnClick and editOnDblclick config option defines whenever editing must start. By default - on element double click.
  • When renaming node - "rename" event is fired. It receives old value as first argument and new value as second argument.
  • You can use zpTreeNodeObj.data.label to access node label or save state of whole Zapatec.Tree using zpTreeObj.saveState() method
  • Also you need to add <script type="text/javascript" src="zapatec/utils/edit_inline.js"></script> to your page.
  • You can finish node editing by pressing Ctrl+Enter
  • Use Tab and Shift-Tab to switch bettwen the nodes while editing.
