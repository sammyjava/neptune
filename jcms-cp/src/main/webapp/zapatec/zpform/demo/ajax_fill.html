Fill form fields using AJAX
  • This demo uses the winxp theme.
  • This form uses the Zapatec Transport module to communicate with the server without refreshing the page.
  • You can automatically fill other form fields depending on typed value. Other field values will be retrieved from server.
  • In current demo fill "Country" field with country name and values for other fields will be retrieved from server
  • To control this use following parameters:
    1. zpFormFillUrl - url where to send request.
    2. zpFormFillMethod - which HTTP method to use (GET, POST etc). The default is GET.
    3. zpFormFillParam - how to name the parameter. The default is the field name.
    4. zpFormFillQuery - string that will be added to the request query.
      5. Example: class='zpFormFillUrl="ajax_fill.php" zpFormFillMethod=GET zpFormFillParam="my_var" zpFormFillQuery="ajax_check=true"' will be transformed to ajax_fill.php?ajax_check=true&my_var=<CURRENT_FIELD_VALUE>
