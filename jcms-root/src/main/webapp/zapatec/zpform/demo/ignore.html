Ignore fields
This form demonstrates how to force Zapatec.Form ignore some fields in the form.
  • This demo uses the lightgreen theme.
  • Sometimes it is useful to make Zapatec.Form to ignore some fields in the form. For example you have some complicated functionality which conflicts with Zapatec.Form functionality.
  • Just add zpIgnoreField to field's CSS class name
