Load Window Content From DIV
  • This demo demonstrates loading the content of the window from a DIV.
  • As you can see here, the DIV can contain, in addition to text, HTML markup including images, links, etc.
  • Note that unlike browser windows, when the user clicks a link inside the window, the whole window page is redirected to that URL (To be able to open links in the same Zapatec Window use iframeContent option, that will put iframe as a content of the Window).
