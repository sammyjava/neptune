Zapatec.Utils.Movable

Zapatec.Utils.Movable(config)
    This is Zapatec.Utils.Movable object definition. This is done in the "interface" manner. It represents most of the routines and events connected to moving of the object.
    Parameters:
      config - {object} - all parameters are passed as the properties of this object. Constructor recognizes the following properties of the config object \code prop. name | description ------------------------------------------------------------------------------------------------- container | {HTML element} The main element to be movable (default null). synchronize | {array} Elements to be synchronize in moving (default []). direction | {string} Enabled direction of moving. Can be "vertical", "horizontal" or "both" | (default "both"). limit | {object} Object with properties "minX", "maxX", "minY" and "maxY", | that are used as restrictions for setting position. moveLayer | {HTML element} Element to be used as movement parent (default document.body). \endcode
