Auto-Complete with Suggestions
  • When you enter any text in a field, the autocomplete feature offers suggestions for the rest of the string.
  • Enter a state name into the AJAX Suggest field. Zapatec will request from the server a suggestion for completing the string. Let's assume that we entered 'Mi' as a value. The server must respond with the JSON object in the following format:
    • {

    "success": true,
    "header": [

		{   
		
			"name": "State name",
			"style": "background-color: #6699CC; color: #FFFFFF"
		}
	],

	"body": [

		["Michigan"],

		["Minnesota"],

		["Mississippi"],

		["Missouri"]
	]

}
  • In case no variants were found starting with letters you provided, the server response will be as follows:
    • {
    "success": true,

	"header": [

		{
			"name": "State name",

			"style": "background-color: #6699CC; color: #FFFFFF"
		}
	],

	"body": [
		
	]
}
  • Enter any value into the Client-side autocomplete field. Zapatec will consult a list of state names to get a suggestion for completing the string.
  • Enter a state name into the Strict client-side autocomplete field. Zapatec will consult a list of options to get a suggestion for completing the string. If you enter a state without initial letter being capitalized or a string that does not match a state name, the field will not be validated.
