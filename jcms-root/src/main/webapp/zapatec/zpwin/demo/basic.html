Basic Windows Example
This example shows several windows open in different locations on the page.
Notice that the windows behave for the most part the way you would expect them to.
  • You can close a window.
  • You can resize a window.
  • You can minimize a window.
  • You can move the focus to a window by clicking its title (to enable activation on clicking anywhere in the window use raiseOnlyOnTitle option). Except first window uses the option and can reach focus by clicking anywhere on it.
You can also programmatically control windows. Click the links below to see how to close and activate windows.
